# 获取商品关键词联想词

#### 接口描述：
- 获取商品关键词联想词。

#### 请求 URL：
- `http|https://host/api/v1/goods/method/get.goods.keywords.suggest/`

#### 请求方式：
- GET
- POST

#### 是否授权：
- 否

#### 业务参数：
|参数名|类型|是否必须|范围值|默认值|示例值|描述|
|:----|:---|:---:|:-----|:-----|:-----|-----|
|keywords |string |是 |max:255 | |华为 笔记本 |搜索关键词 |

#### 响应参数：
|参数名|类型|是否返回|示例值|描述|
|:-----|:-----|:---:|:-----|-----|
|status |integer |是 |200 |状态码 |
|message |string |是 |success |消息信息 |
|data |array |否 |[] |返回数组 |
